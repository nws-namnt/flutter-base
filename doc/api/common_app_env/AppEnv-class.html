AppEnv

AppEnv class

Type-safe wrapper around flutter_dotenv. Usage: await AppEnv.load(flavor: 'dev'); print(AppEnv.apiBaseUrl);

Keys are read from assets/env/.env.<flavor>. Required keys throw StateError at load time if missing or empty.
