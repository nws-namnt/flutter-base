TransitionPage

TransitionPage class

A CustomTransitionPage preconfigured with a PageTransitionType.

Use this as the return value of a GoRoute's pageBuilder:

pageBuilder: (context, state) => TransitionPage(
  child: const MyPage(),
  transitionType: PageTransitionType.slideFromRight,
),

A stable ValueKey is derived from transitionType + child.hashCode so GoRouter does not re-build the page unnecessarily.
